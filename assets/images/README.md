# Images Folder

## Flowman Hero Image

Please add your Flowman superhero image here with the filename:
- `flowman-hero.png` (or `.jpg`, `.webp`, etc.)

The image should be:
- The Flowman superhero character (blue mask, yellow cape, pink and blue costume)
- Recommended size: 400-600px wide
- Format: PNG, JPG, or WebP
- Transparent background preferred (PNG)

The image will be automatically styled with:
- Comic book borders
- Drop shadows
- Yellow accent border
- Subtle floating animation
